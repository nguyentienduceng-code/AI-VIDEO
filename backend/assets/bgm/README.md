# Background Music (BGM)

Thư mục này chứa nhạc nền royalty-free cho video.

## Cách thêm nhạc nền:
1. Đặt file `.mp3` vào thư mục này
2. Đặt tên file theo format: `ten_nhac.mp3` (dùng underscore, không dấu)
3. Hệ thống sẽ tự detect và hiển thị trên giao diện

## Gợi ý tên file (hệ thống sẽ tự format thành tên hiển thị):
- `chill_lofi.mp3` → "Chill Lo-Fi"
- `epic_cinematic.mp3` → "Epic Cinematic"
- `upbeat_pop.mp3` → "Upbeat Pop"
- `soft_piano.mp3` → "Soft Piano"
- `corporate_minimal.mp3` → "Corporate Minimal"

## Nguồn nhạc miễn phí bản quyền:
- https://pixabay.com/music/
- https://www.bensound.com/
- https://freemusicarchive.org/
